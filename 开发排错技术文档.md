## 一、最终可工作的配置

### 硬件连接
```
MCU(STM32F103ZET6)          摄像头模组(FM225)
  PA2 (USART2_TX)    ────    UART_RX
  PA3 (USART2_RX)    ────    UART_TX
  VCC                ────    5.5V~9V ≥1A
  
MCU                       ESP8266-01S
  PB10 (USART3_TX)  ────    RX
  PB11 (USART3_RX)  ────    TX
  3.3V               ────    VCC
  GND                ────    GND

MCU                       电磁锁
  PC3                ────    控制端（高电平开锁）

MCU                       蜂鸣器
  PB8                ────    控制端（高电平响）
```

### 按键功能
| 按键 | MCU引脚 | 功能 | 触发方式 |
|------|---------|------|----------|
| PE3 | PE3 | 录入人脸 | 下拉(GPIO_PIN_RESET) |
| PE2 | PE2 | 验证人脸 | 下拉 |
| PA0 | PA0 | 删除全部用户 | 上拉(GPIO_PIN_SET) |

### 串口分配
| 串口 | 引脚 | 用途 | 波特率 |
|------|------|------|--------|
| USART1 | PA9/PA10 | 下载 HEX（未占用） | - |
| USART2 | PA2/PA3 | FM22x 摄像头模组 | 115200 |
| USART3 | PB10/PB11 | ESP8266 WiFi模块 | 115200 |

## 二、ESP8266 + OneNet MQTT 配置

### WiFi 配置
```c
#define WIFI_SSID       "onenet"
#define WIFI_PASSWORD   "123456789"
```

### OneNet MQTT 配置
```c
#define MQTT_CLIENT_ID  "onenet_one"
#define MQTT_USER       "u9gXgMhYR1"
#define MQTT_PASS       "version=2018-10-31&res=products%2Fu9gXgMhYR1%2Fdevices%2Fonenet_one&et=1812503220&method=md5&sign=ARYnb3UiXlTGHWqufrpm6A%3D%3D"
#define MQTT_HOST       "mqtts.heclouds.com"
#define MQTT_PORT       1883
```

### 云端属性定义
| 属性名 | 类型 | 方向 | 说明 |
|--------|------|------|------|
| LOCK | bool | 双向 | 锁状态（true=开锁, false=关锁） |
| EERNUM | bool | 设备→云端 | 警报数据（true=识别成功, false=识别失败） |
| YC_BJ | bool | 云端→设备 | 远程报警（true=触发蜂鸣器响3下） |
| OPEN_JC | bool | 云端→设备 | 开启人脸验证 |
| OPEN_LR | bool | 云端→设备 | 开启人脸录入 |
| Clear_user | bool | 云端→设备 | 清除所有用户 |
| USER_NUM | int | 设备→云端 | 已录入用户数量 |

### AT 指令格式参考
```c
// WiFi 连接
AT+CWJAP="onenet","123456789"

// MQTT 配置（token 中的 % 需要转义为 %%）
AT+MQTTUSERCFG=0,1,"onenet_one","u9gXgMhYR1","version=2018-10-31&res=products%%2Fu9gXgMhYR1%%2Fdevices%%2Fonenet_one&et=1812503220&method=md5&sign=ARYnb3UiXlTGHWqufrpm6A%%3D%%3D",0,0,""

// MQTT 连接
AT+MQTTCONN=0,"mqtts.heclouds.com",1883,1

// 上报属性（多个属性用 \, 分隔）
AT+MQTTPUB=0,"$sys/u9gXgMhYR1/onenet_one/thing/property/post","{\"id\":\"123\",\"version\":\"1.0\",\"params\":{\"LOCK\":{\"value\":true},\"EERNUM\":{\"value\":true}}}",0,0
```

## 三、发现并修复的所有问题

### 问题 1：lock_state 重复定义
- **现象**：编译报错 Symbol multiply defined
- **修复**：fm22x.c 改为 `extern uint8_t lock_state;`

### 问题 2：ESP8266 初始化缺失
- **现象**：AT 指令无响应
- **修复**：
  1. 添加 ESP8266_Init() 调用
  2. 添加 AT+RST 复位
  3. 添加 AT+CWMODE=1 设置模式

### 问题 3：MQTT Token 转义问题
- **现象**：Token 中的 % 被 sprintf 转义
- **修复**：MQTT Config 使用 strcat 拼接，不用 sprintf

### 问题 4：云端 set_reply 超时
- **现象**：设备不回复 set_reply，云平台等待6秒后超时
- **修复**：收到指令后必须调用 OneNet_SendSetReply()

### 问题 5：NID_FACE_STATE 字段顺序不一致
- **现象**：官方手册与实际固件字段顺序不同
- **修复**：以实际数据自洽性为准

### 问题 6：EnrollSingle 不支持
- **现象**：此摄像头固件只支持交互式录入
- **修复**：使用 FM22x_Enroll(0, 0x1F, 30)

## 四、常见失败原因速查

| 错误码 | 原因 | 解决 |
|--------|------|------|
| 13 TIMEOUT | 人脸不合格/未检测到 | 调整姿态、转动头部 |
| 10 ENROLLED | 人脸已录入 | 先删除再录入 |
| 9 MAXUSER | 100人已满 | 删除部分用户 |
| 4 CAMERA | 摄像头故障 | 检查连接、供电 |
| 8 UNKNOWNUSER | 未录入用户 | 先录入再验证 |

### 供电要求
```
FM225 摄像头模组：5.5V~9.0V，≥1A
ESP8266-01S：3.3V，≥500mA
电磁锁：5V~12V，≥1A
```
