## 1. 硬件连接

```
MCU STM32F103ZET6          摄像头 FM225
  PA2 (USART2_TX)    ────  UART_RX
  PA3 (USART2_RX)    ────  UART_TX
  VCC                ────  5.5V~9V ≥1A

MCU                       ESP8266-01S
  PB10 (USART3_TX)  ────  RX
  PB11 (USART3_RX)  ────  TX
  3.3V               ────  VCC
  GND                ────  GND

MCU                       电磁锁
  PC3                ────  控制端（高电平开锁）

MCU                       蜂鸣器
  PB8                ────  控制端（高电平响）
```

| 串口 | 引脚 | 波特率 | 用途 |
|------|------|--------|------|
| USART1 | PA9/PA10 | - | 下载 HEX（未占用） |
| USART2 | PA2/PA3 | 115200 | 摄像头通信 |
| USART3 | PB10/PB11 | 115200 | ESP8266 WiFi |

## 4. 云端功能

### OneNet MQTT 配置
```c
#define WIFI_SSID       "onenet"
#define WIFI_PASSWORD   "123456789"
#define MQTT_CLIENT_ID  "onenet_one"
#define MQTT_USER       "u9gXgMhYR1"
#define MQTT_HOST       "mqtts.heclouds.com"
#define MQTT_PORT       1883
```

### 云端属性定义
| 属性名 | 类型 | 方向 | 说明 |
|--------|------|------|------|
| LOCK | bool | 双向 | 锁状态（true=开锁, false=关锁） |
| EERNUM | bool | 设备→云端 | 警报数据（true=识别成功, false=识别失败） |
| YC_BJ | bool | 云端→设备 | 远程报警（true=触发蜂鸣器响3下） |
| OPEN_JC | bool | 云端→设备 | 开启人脸验证 |
| OPEN_LR | bool | 云端→设备 | 开启人脸录入 |
| Clear_user | bool | 云端→设备 | 清除所有用户 |
| USER_NUM | int | 设备→云端 | 已录入用户数量 |

### 云端下发指令处理
```c
// main.c 主循环
if (mqtt_connected) {
    char cmd_id[32] = {0};
    int8_t cloud_cmd = OneNet_CheckCommand(cmd_id);

    switch (cloud_cmd) {
        case 0: do_lock(); break;           // LOCK: false
        case 1: do_unlock(); break;         // LOCK: true
        case 2: FM22x_Verify(1, 10); break; // OPEN_JC
        case 3: FM22x_Enroll(0, 0x1F, 30); break; // OPEN_LR
        case 4: FM22x_DeleteAll(); break;   // Clear_user
        case 5: /* 蜂鸣器响3下 */ break;    // YC_BJ
    }
    if (cmd_id[0]) OneNet_SendSetReply(cmd_id);
}
```

## 5. 数据流架构

```
摄像头 UART_TX
    │ 115200bps
    ▼
MCU PA3 (USART2_RX) ──→ FM22x_UART_Callback()
    │                     └─ 状态机解析帧
    │                         └─ ProcessReceivedFrame()
    │
    └─ USART3 (ESP8266)
         └─ rx_buf[] 缓存 MQTT 消息

main() 循环 (100ms)
    ├─ 人脸状态显示
    ├─ 命令结果处理（录入/验证）
    ├─ 按键检测
    ├─ 云端指令检查
    └─ 心跳上报(30s)
```

## 6. 协议帧格式

```
EF AA | MsgID | SizeH SizeL | Data[0..N] | Checksum
 2B   |  1B   |  2B(大端)   |   N字节     |  1B

校验: Checksum = MsgID ^ SizeH ^ SizeL ^ Data[0] ^ ... ^ Data[N-1]
```

### 命令速查
| 命令 | Code | 用法 |
|------|------|------|
| VERIFY | 0x12 | `FM22x_Verify(1, 10)` |
| ENROLL | 0x13 | `FM22x_Enroll(0, 0x1F, 30)` |
| DELETEALL | 0x21 | `FM22x_DeleteAll()` |
| GET_ALL_USERID | 0x24 | 查询用户数量 |

## 7. 操作说明

### 录入 (PE3)
1. 正对摄像头 30-50cm
2. 按 PE3
3. **缓慢大幅度转头**：上→下→左→右→正
4. 每个方向停 2 秒
5. 30 秒内完成

### 验证 (PE2)
1. 正对摄像头
2. 按 PE2
3. 成功 → 开锁 + 上报云端
4. 失败 → 蜂鸣器报警 + 上报云端

### 云端控制
| 指令 | 功能 |
|------|------|
| OPEN_JC: true | 触发人脸验证 |
| OPEN_LR: true | 触发人脸录入 |
| Clear_user: true | 清除所有用户 |
| LOCK: true | 开锁 |
| LOCK: false | 关锁 |
| YC_BJ: true | 远程报警（蜂鸣器响3下） |

## 8. 常见错误码

| Code | 含义 | 解决 |
|------|------|------|
| 0 | 成功 | - |
| 8 | 无匹配用户 | 先录入再验证 |
| 9 | 用户已满(100) | 删除部分用户 |
| 10 | 人脸已录入 | 删除后重新录入 |
| 12 | 活体检测失败 | 确保真人面对摄像头 |
| 13 | 超时 | 调整姿态/转头/光线 |
